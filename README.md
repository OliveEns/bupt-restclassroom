# bupt-restclassroom

北邮主楼空闲教室分析油猴脚本，自动登录北邮移动教务系统并生成可视化的空闲教室表格。

## 🤖 AI生成提醒

- 本项目源代码及文档由AI辅助生成

## 📋 功能说明

1. **自动登录**：配置后自动填写北邮移动教务系统账号密码并登录
2. **智能导航**：自动点击"今日空闲教室"按钮进入查询页面
3. **数据获取**：自动拦截并获取空闲教室数据
4. **智能分析**：
   - 计算每个教室的空闲时段
   - 按空闲时间排序（全天空闲 > 连续至最后时段 > 最长连续时长）
   - 统计全天空闲教室数量
   - 找出最长连续空闲段
   - 推荐最优空闲教室
5. **可视化展示**：
   - 以表格形式直观展示各教室空闲状态
   - 支持横纵向滚动查看全部信息
   - 响应式设计，适配不同屏幕尺寸
   - 美观的界面样式和交互效果

## 🚀 使用方法

### Windows Edge浏览器

1. **安装油猴扩展**：
   - 打开Edge浏览器
   - 访问Microsoft Store，搜索并安装"Tampermonkey"
   - 安装完成后，浏览器右上角会出现油猴图标

2. **添加脚本**：
   - 点击油猴图标，选择"添加新脚本"
   - 清空编辑器中的默认内容
   - 将main.js文件的完整内容复制粘贴到编辑器中
   - 修改脚本中的登录配置（第15-16行）：
     ```javascript
     username: "你的学号",
     password: "你的教务系统密码",
     ```
   - 点击编辑器左上角的"文件"，选择"保存"
   - 在浏览器“设置-扩展”中打开开发者模式

3. **使用脚本**：
   - 打开北邮移动教务系统登录页面：`http://jwglweixin.bupt.edu.cn/sjd/`
   - 脚本会自动执行登录操作
   - 登录成功后，自动点击"今日空闲教室"按钮
   - 等待片刻，系统会显示优化后的空闲教室表格

### 移动端Via浏览器

1. **安装Via浏览器**：
   - 在应用商店搜索并安装"Via浏览器"
   - 打开Via浏览器，点击底部工具栏的三个横线图标

2. **配置脚本功能**：
   - 选择"设置"，进入设置页面
   - 找到并点击"脚本"
   - 开启"启用脚本"
   - 点击"添加脚本"

3. **添加脚本**：
   - 将main.js文件的完整内容复制粘贴到脚本编辑器中
   - 修改脚本中的登录配置（第15-16行）：
     ```javascript
     username: "你的学号",
     password: "你的教务系统密码",
     ```
   - 点击右上角的"保存"

4. **使用脚本**：
   - 在Via浏览器中访问北邮移动教务系统登录页面：`http://jwglweixin.bupt.edu.cn/sjd/`
   - 脚本会自动执行登录操作
   - 登录成功后等待片刻，系统会显示优化后的空闲教室表格

## ⚠️ 注意事项

1. 请确保学号和密码正确，否则无法自动登录
2. 请定期检查脚本是否正常工作，系统更新可能导致脚本失效
3. 请勿将包含个人账号密码的脚本分享给他人

## 📊 界面说明

- **标题区域**：显示当前页面为"北邮主楼空闲教室分析"及数据更新时间
- **统计信息栏**：显示教室总数、全天空闲教室数、最长连续空闲段、最优空闲教室等信息
- **空闲教室表格**：
  - 左侧第一列为教室名称
  - 顶部为时间段（节次）
  - 绿色"✓"表示空闲，红色"✗"表示忙碌
  - 教室按空闲时间排序，优先展示全天空闲或连续空闲时间最长的教室
